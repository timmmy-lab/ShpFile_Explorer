# How to run this program

Due to OpenStreetMap attribution policy, you cannot run this file directly from your file system (using `file:///`)

To run this file in your browser, you first must execute

`python -m http.server 8000`

then in your browser address bar navigate to

`http://localhost:8000/path/to/index.html`

For example, if you have your file stored in Downloads:

`http://localhost:8000/Downloads/index.html`
